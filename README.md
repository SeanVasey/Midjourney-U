## About

Midjourney-U (for University) is an easily digestible, self-paced tutorial companion guide that helps users make the most of the awesome power Midjourney can unleash on the quality of your output.

Whether you're just getting started or looking to refine advanced techniques, Midjourney-U provides structured lessons, practical examples, and clear explanations to accelerate your creative workflow.

## Features

### Learning

- Structured, progressive tutorial modules
- Real-world prompt examples and breakdowns
- Tips for getting consistent, high-quality results

### Reference

- Parameter and command cheat sheets
- Style and aesthetic guides
- Workflow best practices
